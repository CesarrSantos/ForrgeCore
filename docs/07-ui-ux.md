# UI / UX

- Centro: minijuego
- Arriba: recursos
- Derecha: upgrades rápidos
- Izquierda: botón Árbol
- Abajo: Ascensión

Diseño pensado para desktop y móvil.

## Posibles mejoras de UX (cola de tareas)
- Feedback reforzado en golpes (destellos, sacudida leve, sonido)
- Indicadores de ganancias por golpe y por segundo
- Estado de combo visible con barra/medidor y degradado
- Tooltips y microcopys para cada upgrade y recurso
- Indicador de coste/beneficio en upgrades (ROI simple)
- Animación de incremento en contadores y monedas que vuelan al HUD
- Panel compacto para versión móvil (apilar tarjetas + menú flotante)
- Indicador de guardado y tiempo offline estimado
- Tema claro/oscuro con conmutador rápido
