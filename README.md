# Forja del Núcleo

Idle / incremental con minijuego de timing. Implementado con HTML + JavaScript + Phaser.

## Inicio rápido

- Abre `index.html` en un servidor local (recomendado) o directo en el navegador.
- Linux/macOS: ejecuta `./run.sh`.
- Windows: ejecuta `run.bat`.

Si quieres cambiar el puerto, define `PORT` antes de ejecutar el script (por ejemplo, `PORT=8080 ./run.sh` en Linux/macOS o `set PORT=8080` antes de `run.bat` en Windows).

## Dependencias

No hay dependencias de build. Solo necesitas Python 3 para levantar el servidor local (los scripts te avisan si falta).

## Estructura

- `src/` código del juego
- `assets/` arte y audio
- `docs/` documentos de diseño y tareas
